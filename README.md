# Vagrant-images
**VM automation**

Tools required:

1. vmware

To install Vagrant (MacOS)

`brew install vagrant`

* Steps to use: *

Step 1:

`vagrant up`

Step 2: wait until the vmware pops up and does the necessary steps automatically 

`vagrant ssh`

Now you are in the ubuntu bash

* To destroy a vangrant *

` vagrant destroy  `
